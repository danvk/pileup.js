# pileup.js
Interactive in-browser track viewer

## Quickstart

  git clone https://github.com/danvk/pileup.js.git
  cd pileup.js
  npm install
  tsd reinstall
  grunt prod

Then open `playground.html` in your browser of choice.

## Development

Run the tests:

  grunt test

Iterate on the JS:

  grunt watch
